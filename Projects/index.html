    Projects

    • Education Management Information System for an International NGO Microfinance Program Full-stack MIS (React, TypeScript, Django REST, PostgreSQL) supporting participant workflows, Excel import, role-based access, and an executive KPI dashboard, with AI-assisted suggestions for operational decision-making.
    • Intelligent API Monitoring and Alert System Node.js/Express service that ingests API response metrics, scores anomalies, and uses Groq (LLaMA) to produce plain-English alerts stored in MongoDB; optional email notifications and a live dashboard for incident tracking.
    • Browser-Based Human–Robot Interaction Platform for a Differential-Drive Robot (ROS 2 Jazzy) Django REST and rclpy bridge for teleoperation, SLAM mapping, AMCL localization, and Nav2; React/Vite frontend for live map visualization and multi-user monitoring.
    • ESP32 Firmware for a Differential-Drive Robot Low-level firmware for Buddy with L298N motor drive, quadrature encoder odometry, and a ROSArduinoBridge serial interface; pairs with diffdrive_arduino / ros2_control on the ROS 2 host for real-time motor control and odometry.
    • Buddy — ROS 2 Jazzy Robotics Stack (URDF, Nav2, Teleop, ros2_control) ROS 2 mobile robotics platform integrating SLAM, AMCL, and Nav2 for autonomous navigation, with ESP32-based motor control, encoder feedback, ros2_control, and a web interface for teleoperation and monitoring (private repository available upon request).
    • Optimized Waste Truck Routes in Ghana Field deployment with CIS Tech Ltd. and the Government of Ghana: operational data analysis to replan collection routes, reduce fuel consumption, and deliver real-time notifications via a mobile application.
    • Library Management System API Django REST Framework backend with user registration, role-based access (student/staff), book catalog management, and rent/return workflows.
    • Autonomous Indoor Navigation Car Using ROS 1 Melodic Mobile robot on Ubuntu 18.04 with Jetson Nano and Arduino integration, supporting SLAM, localization, and autonomous navigation via the ROS Navigation Stack with LiDAR and encoder-based control.
